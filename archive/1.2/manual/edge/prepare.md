## 边缘节点配置要求

### 主机资源配置

| CPU(核) | 内存(G) | 系统盘(G) | 数据盘(G) |
| ------- | ------- | --------- | --------- |
| > 4      | > 8G     | 40G SSD   | 200G SSD  |

### 操作系统要求

| Linux 操作系统平台       | 版本       |
| :----------------------- | :--------- |
| Red Hat Enterprise Linux | 8.2 及以上 |
| CentOS                   | 8.2 及以上 |

# 选择部署模式

Erda 支持两种部署模式：**私有化**、**SaaS 化**。

- 私有化：是指部署一个完整的 Erda 平台，包含 Erda 所有的组件。
- SaaS 化：是一种以托管的方式在客户测部署一个 Erda 集群，将该集群统一由 Erda 中心平台进行纳管。客户会通过统一的 Internet 域名入口进行 Erda 访问。
